# Multi-LLM Web Orchestrator

Chrome extension for comparing and routing prompts across multiple AI providers.

## Features

- Side panel interface
- Multi provider support
- Prompt comparison
- Local API key storage
- Response history
- TypeScript

## Install

```bash
npm install
```

Run:

```bash
npm run dev
```

Build:

```bash
npm run build
```

Load extension:

Chrome →
Extensions →
Developer Mode →
Load unpacked →
dist
